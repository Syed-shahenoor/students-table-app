# 🎓 Students Table App

A simple web application built using **HTML**, **CSS**, and **JavaScript** to manage student records in a table. The application allows users to add, edit, and delete student information through an interactive interface.

## Features

- Add new student records
- Display student details in a table
- Edit existing records
- Delete student records
- Interactive and user-friendly interface

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

```
students-table-app/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Installation

### Clone the repository

```bash
git clone https://github.com/Syed-shahenoor/students-table-app.git
```

### Navigate to the project directory

```bash
cd students-table-app
```

### Run the project

Open `index.html` in your preferred web browser.

## Usage

1. Open the application.
2. Enter the student details in the form.
3. Click **Add** to insert a new record into the table.
4. Use the **Edit** option to modify a student's details.
5. Use the **Delete** option to remove a record.
